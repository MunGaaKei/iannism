HTML 节点属性

记录及说明了部分个人觉得比较有帮助的HTML节点属性，持续更新。

accesskey

通过快捷键访问该节点，该节点需具备可通过TAB获取焦点的条件，可通过Alt+k进行获取焦点:

    
<input type="text" accesskey="k" placeholder="press key Alt + K">

contentEditable

将节点内容变为可编辑状态，通常用于开发富文本编辑器。可选并且常用的值有：

  • contenteditable:
    富文本形式
  • plaintext-only:
    纯文本输入

enterKeyHint

屏幕键盘中Enter键显示的文本。

tabIndex

当值大于或等于0时，节点可以被Tab键索引，一般默认值为-1。
