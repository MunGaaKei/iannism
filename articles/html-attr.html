HTML 节点属性

记录及整理了部分个人觉得比较有帮助的HTML节点属性，持续更新。

accesskey

通过快捷键访问该节点，该节点需具备可通过TAB获取焦点的条件，可通过Alt+k进行获取焦点:

    
<input type="text" accesskey="k" placeholder="press key Alt + K">

enterKeyHint

屏幕键盘中Enter键显示的文本。

autocomplete

一般情况下，遇到相同name值的输入框时，浏览器会将之前记录的填入值以类似下拉框的形式显示出来，以便你可以迅速填入，默认为on开启。

如果想要清除其中某个候选值，可以使用↑↓键选中，然后按下Shift+Del键。Mac系统还需多按一个fn键。最新版Edge浏览器则会出现对应的按钮。

可选的值有：

  • on:
    开启
  • off:
    关闭

点击这里切换下面输入框的自动填写开启关闭。

    
<input type="text" name="name" autocomplete="on">

contentEditable

将节点内容变为可编辑状态，通常用于开发富文本编辑器。可选并且常用的值有：

  • contenteditable:
    富文本形式
  • plaintext-only:
    纯文本输入

disabled

常用于禁用表单控件，可搭配CSS:disabled选择器使用：

    
input:disabled{
    background: #ffecec;
    cursor: not-allowed;
}

enterKeyHint

屏幕键盘中Enter键显示的文本。

list

与autocomplete的功能一样，通过这个属性可以设置输入框的推荐填入值列表，与<datalist>标签搭配使用：

    
<input type="text" list="list-recommand" name="name">
<datalist id="list-recommand">
    <option value="Jesse Lee">Jesse</option>
    <option value="Stephen Chow">Stephen</option>
    <option value="Jay Chow">Jay</option>
</datalist>

oncontextmenu

设置一个节点的鼠标右键按钮事件，一般常用于禁止右键点击弹出上下文。用法如下：右键点击我将不会弹出上下文窗口。

    
<a oncontextmenu="return false;">右键点击我</a>

spellcheck

检查拼写，一般仅对输入框有效果（红色波浪线提示），默认为true开启。

可选的值有：

  • true:
    开启
  • false:
    关闭

如果觉得影响美观，点击这里可以切换下面输入框的拼写检查开启关闭。

    
<input type="text" spellcheck value="spellchcek">

tabIndex

接收一个整数作为值，节点可以被聚焦和Tab键索引。

title

不止<a>标签，其它节点也可以设置这个属性，作用一样，在鼠标悬停的时候可以显示一些信息。请将鼠标悬停在这里。
