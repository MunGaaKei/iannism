方法选择元素CSS
div+p匹配<div>后的邻居<p>元素2
span~b匹配前面有<span>的每个<b>元素3
p:not(:first-child)非第一个节点3
p:nth-child(3n+1)第1、4、7、11…个<p>元素3
p:nth-last-child(3)属于父元素里倒数第3个<p>元素3
p:nth-of-type(3)同一节点层第三个<p>元素3
p:empty匹配空节点的<p>元素3
p:only-child属于父元素里只含有<p>元素的这个<p>元素3
p:[class|=blue]匹配类名以blue-**这个类格式开头的<p>元素3
p:[class~=blue]匹配类名含有blue的<p>元素3
p:[class*=blue]匹配类名里含有blue字符串的<p>元素3
img:[src^=http]匹配src以http为开头的<img>元素3
img:[src$=jpg]匹配src以jpg为结尾的<img>元素3



## p:[class=a]匹配不到<p class="a b"> 2 </p>元素，p:[class="a b"]才可以匹配得到。




## nth-child() 与 nth-of-type() 的区别


<div>
　<b> 1 </b>
　<p> 2 </p>
　<p> 3 </p>
</div>
在如上HTML结构中

p:nth-child(2){ color: red; } 会让第一个<p>元素也就是数字2字体变红。
p:nth-of-type(2){ color: red; } 则会让第二个<p>元素也就是数字3变红。

## 要注意的是，如果第二个元素不是<p>元素，那么p:nth-child(2)则不会匹配到任何元素。
