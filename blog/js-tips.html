匿名事件监听绑定及移除

此方法可以用在类似实现绑定匿名事件只让它执行一次的功能下（以及懒得重新声明一个事件句柄来放入removeEventListener的第二个参数）：
el.addEventListener('click', function(){

	// do something ...

	this.removeEventListener('click', arguments.callee);

});
ps：此方法在 "use strict"; 下无效。




单句表达式判断执行

// 通常方式
if (condition) eval();

// 另一种方式
condition && eval();




假值表

var  a = false,		// 传统布尔值 false
	b = 0,			// 数字0
	c = '',			// 空字符串
	d = 1 * 'd',	      // NaN类型
	e;				// 未赋值变量 undefined

假值表上的变量在判断时都会隐性转换为false。

* 非空对象和空对象 var f = { } 转成布尔值都为true，所以不能用 if ( f ) 来判断对象是否为空。可以用 "{}" == JSON.stringify( f ) 来判断对象是否为空。
